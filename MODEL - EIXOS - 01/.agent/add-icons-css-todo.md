# Script para adicionar custom-icons.css em todas as páginas

## Páginas que JÁ TÊM:

- ✅ index.html
- ✅ quem-somos.html
- ✅ icons-catalog.html

## Páginas que PRECISAM:

- ⏳ projetos.html
- ⏳ galeria.html
- ⏳ noticias.html
- ⏳ doe-aqui.html
- ⏳ fale-conosco.html
- ⏳ editais.html
- ⏳ todos-projetos.html

## Ação:

Adicionar antes do `</head>`:

```html
<link rel="stylesheet" href="assets/css/custom-icons.css" />
```
